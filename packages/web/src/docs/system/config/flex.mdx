# Borders

```ts
import { flexSystem } from "@poodle/system";
```

`flex` system support CSS Flexbox.

```ts

const listSystem = compose(flexSystem);

export const system = createSystem(listSystem);

const styles = system({
	alignItems: "flex-start",
	justifySelf: "center",
});

// Styles result
//  { alignItems: "flex-start" },
//  { justifySelf: "center" },
```
