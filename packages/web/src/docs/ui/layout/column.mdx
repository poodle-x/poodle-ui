| Name     | Type                               | Description                                                                             |
| -------- | ---------------------------------- | --------------------------------------------------------------------------------------- |
| children | `React.ReactNode`                  |                                                                                         |
| colWidth | `ResponsiveProp<string \| number>` | Set column width. If we want the column width equal to content set `colWidth="content"` |
