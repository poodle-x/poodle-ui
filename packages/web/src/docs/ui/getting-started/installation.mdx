# @poodle/ui


## Installation

With NPM

```bash
npm install @emotion/css @emotion/react @emotion/styled @poodle/ui
```

With Yarn

```bash
yarn add @emotion/css @emotion/react @emotion/styled @poodle/ui
```

After that check the document for [usage](/ui/getting-started/usage).
