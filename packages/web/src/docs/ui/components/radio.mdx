# Radio

```jsx
import Radio from "@poodle/ui/Radio";
```

`Radio` is a component allow the user to select a value from a list of options.

## Simple

Each Radio must have a label for accessibility. We can hide the displayed label by using prop `isLabelHidden={true}`

```jsx-live
  <Box>
    <Box>
      <Radio isLabelHidden={true} label="Hidden label"  />
    </Box>
    <Box mt="scale-2">
      <Radio defaultChecked={true} label="With Label" />
    </Box>
  </Box>
```

## Sizes

Change Radio size via prop `sizeStyle`.

Default Radio sizes are `s`, `m`, `l`. If you do not set size, the Radio will use the size `m` as default.

```jsx-live
  <Box>
    <Box mt="scale-2">
      <Radio label="Small" defaultChecked={true} sizeStyle="s" />
    </Box>
    <Box mt="scale-2">
      <Radio label="Medium" defaultChecked={true} sizeStyle="m" />
    </Box>
    <Box mt="scale-2">
      <Radio label="Large" defaultChecked={true} sizeStyle="l" />
    </Box>
  </Box>
```

### Local Props


| Name           | Type      | Description                                                                                          |
| -------------- | --------- | ---------------------------------------------------------------------------------------------------- |
| **label \* **  | `string`  | Label of the Radio.                                                                                  |
| defaultChecked | `boolean` | Change Radio default checked.<br />Will override input props `defaultChecked`.                       |
| isChecked      | `boolean` | Change Radio to checked state when set to **true**.<br />Will override input props `checked`.        |
| isDisabled     | `boolean` | Change Radio icon to disabled state when set to **true**.<br />Will override input props `disabled`. |
| isLabelHidden  | `boolean` | Hide label when set to **true**                                                                      |
