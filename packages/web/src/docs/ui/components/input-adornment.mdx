# Input Adornment

```jsx
import Input from "@poodle/ui/InputAdornment";
```

`InputAdornment` is a component use together with `Input` component.

Please check `Input` [document](/ui/components/input) to see the example usage.

### Local Props


| Name              | Type               | Description                                                                           |
| ----------------- | ------------------ | ------------------------------------------------------------------------------------- |
| adornmentPosition | `"start" \| "end"` | Change position styles when the adornment is the first item<br />at the begin or end. |
| children          | `React.ReactNode`  |                                                                                       |
| isIcon            | `boolean`          | Change to icon styles when the adornment content is a icon.                           |
